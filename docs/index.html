FedCast

Federated Learning for Time Series Forecasting

A comprehensive Python framework designed for time series forecasting using federated learning. Built on top of the powerful Flower framework to enable privacy-preserving, decentralized model training.

Quick Start

# Install FedCast
pip install fedcast

# Import and use
from fedcast.datasets import SinusDataset
from fedcast.cast_models import MLP
from fedcast.federated_learning_strategies import FedTrend

# Load time series data
dataset = SinusDataset(num_clients=10, sequence_length=100)

# Define model architecture
model = MLP(input_size=100, hidden_size=64, output_size=1)

# Choose federated learning strategy
strategy = FedTrend()

# Run federated learning experiment
results = run_federated_experiment(
    dataset=dataset,
    model=model,
    strategy=strategy,
    num_rounds=50
)

Note: FedCast is currently in Beta (v0.1.1b1). Core functionality is stable, but some features may still be under development.
